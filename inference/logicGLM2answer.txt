解题思路:

根据无向图的性质,每条边连接的两个结点之间的权值应该相等。因此,我们可以将每条边连接的两个结点之间的权值都设为1,那么有15条边,所以总权值为15。

接下来,我们需要考虑每个4度结点对总权值的影响。每个4度结点会使总权值增加4,因为每个4度结点连接的两个结点之间的权值为4。因此,有3个4度结点,那么总权值增加3*4=12。

从根结点开始,以当前结点的度数递归地遍历所有子结点。当遍历到一个3度结点时,回溯到上一个结点,继续遍历。当遍历到一个叶子结点时,输出该结点的度数并返回。

根据这个思路,我们可以得到如下的代码实现:

```
def dfs(node, level=0):
    if node.is_leaf:
        return node.度数
    else:
        for child in node.neighbors:
            dfs(child, level+1)

def tree_height(root):
    height = 0
    if not root.is_leaf:
        for child in root.neighbors:
            height = max(height, dfs(child, level=0))
    return height
```
